🚀 Ανέβασμα site με GitHub Pages (βήμα-βήμα)
Τι θα χρειαστείς

Έναν λογαριασμό GitHub (δωρεάν)

Ένα αρχείο index.html

10–15 λεπτά ⏱️

1️⃣ Δημιούργησε λογαριασμό στο GitHub

Πήγαινε: github.com
→ Sign up
(βάζεις email, username, password)

2️⃣ Δημιούργησε νέο repository

Αφού συνδεθείς:

Πάτα New repository

Βάλε:

Repository name: π.χ. my-first-site

Public ✅

Πάτα Create repository

3️⃣ Δημιούργησε το αρχείο index.html

Μέσα στο repository:

Πάτα Add file → Create new file

Όνομα αρχείου:

index.html


Βάλε μέσα αυτό (παράδειγμα):


<!DOCTYPE html>
<html>
  <head><title>Τα ζώα της Αφρικής</title>
    </head>
  <body><h1>Γεια σου κόσμε!</h1>
  <p>Αυτό είναι το site μου στο GitHub Pages 🎉</p><h1>Ο Βασιλιάς της Σαβάνας</h1>
<img src="https://upload.wikimedia.org/wikipedia/commons/thumb/7/73/Lion_waiting_in_Namibia.jpg/640px-Lion_waiting_in_Namibia.jpg" alt="Λιοντάρι στη Σαβάνα" width="400">
<p>Το λιοντάρι είναι ένα από τα μεγαλύτερα αιλουροειδή και ζει κυρίως στην Αφρική. Είναι γνωστό για την εντυπωσιακή χαίτη του αρσενικού και θεωρείται σύμβολο δύναμης.</p>
<a href="https://wikipedia.org">ΠΕΡΙΣΣΟΤΕΡΑ στο WIKIPEDIA</a>
    </body>
</html>



Πάτα Commit new file

4️⃣ Ενεργοποίησε το GitHub Pages

Πήγαινε Settings (στο repository)

Από αριστερά → Pages

Στο Source:

Branch: main

Folder: /root

Πάτα Save

5️⃣ Το site σου είναι online 🌍

Σε λίγα δευτερόλεπτα θα δεις μήνυμα:

Your site is published at:
https://ΤΟ-USERNAME.github.io/my-first-site/


👉 Άνοιξε το link και έτοιμο!
